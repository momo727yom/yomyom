
# 사주팔자 애플리케이션 블루프린트

## 개요

이 애플리케이션은 사용자가 자신의 생년월일을 입력하면 오늘의 운세를 알려주는 간단한 웹 애플리케이션입니다.

## 기능

*   **생년월일 입력:** 사용자는 년, 월, 일을 입력할 수 있습니다.
*   **운세 보기:** "운세 보기" 버튼을 클릭하면 오늘의 운세를 볼 수 있습니다.
*   **운세 표시:** 입력된 생년월일을 기반으로 생성된 운세가 화면에 표시됩니다.

## 디자인 및 스타일

*   신비로운 느낌을 주기 위해 어두운 테마를 사용합니다.
*   가독성을 높이기 위해 명확하고 읽기 쉬운 글꼴을 사용합니다.
*   사용자 인터페이스는 간단하고 직관적으로 구성됩니다.

## 구현 계획

1.  **HTML (`index.html`):**
    *   페이지 제목을 "사주팔자"로 변경합니다.
    *   "오늘의 운세"라는 제목의 헤더를 추가합니다.
    *   생년월일 입력을 위한 폼(년, 월, 일)을 생성합니다.
    *   운세 결과를 표시할 섹션을 추가합니다.
2.  **CSS (`style.css`):**
    *   어두운 배경색과 대비되는 텍스트 색상을 설정합니다.
    *   페이지 요소들의 레이아웃과 간격을 조정합니다.
    *   버튼과 입력 필드에 스타일을 적용합니다.
3.  **JavaScript (`main.js`):**
    *   사용자가 "운세 보기" 버튼을 클릭했을 때 실행될 함수를 작성합니다.
    *   입력된 생년월일 데이터를 가져옵니다.
    *   간단한 운세 생성 로직을 구현합니다. (예: 생년월일 숫자를 조합하여 무작위 운세 메시지 선택)
    *   생성된 운세를 HTML의 결과 섹션에 표시합니다.
